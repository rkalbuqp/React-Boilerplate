# Welcome to Your New Project

This project is set up with:
- **React + TypeScript + Vite**
- **Storybook** for UI development
- **Vitest** for testing
- **ESLint + Prettier** for code quality

## Structure

- `src/components`: Reusable UI components
- `src/pages`: Application pages (routed)
- `src/hooks`: Custom React hooks
- `src/utils`: Utility functions
- `src/styles`: Global styles and theme
- `src/types`: TypeScript type definitions
- `src/services`: API calls and external services
- `src/contexts`: React Context definitions

## Getting Started

Start adding your components in `src/components` and create stories for them to see them here!
